# SSHIP

**SSHIP** manage your ssh keys hassle free

## ✨ Features
- SSH gen
- SSH backup
- SSH session helpers
- Quick config creation
- more ...

## ⚡ Quick Install



Install `sship` using `curl`:

```bash
curl -sSL https://raw.githubusercontent.com/Makumiii/sship/main/install.sh | bash
```


## To run tool

```bash
sship
```
